This page lists my key publications in AI/ML for Wireless Communications, CSI-based sensing, mmWave/6G systems, and multimodal HAR. A complete and always-updated list is available on my ResearchGate profile.

Journal & Conference Publications

Multi-Link Prediction for mmWave Wireless Communication Systems Using Liquid Time-Constant Networks, Long Short-Term Memory, and Interpretation Using Symbolic Regression

Milind Patil, et al.

IEEE ANTS / 2023–2024

A hybrid AI approach combining LTC networks, LSTM models, and symbolic regression to interpret and forecast mmWave multi-link quality for 5G/6G systems.

Human Activity Recognition (HAR) Using CSI + Vision with LLM/VLM Fusion

Milind Patil, et al.

Submitted to IEEE ANTS 2025

A multimodal HAR system integrating CSI-based LLM models with vision transformers, enabling robust fall detection and activity recognition using timestamp-aligned sensory fusion.

UAV Path Planning Using CNN-based Scene Understanding

Milind Patil, et al.

MDPI Sensors (Under Review)

Proposes a deep learning–based framework that integrates CNN-driven visual scene understanding with UAV navigation for safe and autonomous flight path generation.
